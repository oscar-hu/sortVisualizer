# Sort Visualizer
Visualization on popular sorting algorithms such as Insertion Sort, Selection Sort, Merge Sort, Quick Sort, etc. Includes a GUI made using Tkinter library with descriptive highlights to emphasize selection process in sorting algorithm. 

# How to Install
On the command line,
- `git clone https://github.com/oscar-hu/sortVisualizer.git`
- `cd sortVisualizer/`
- Depending on your version of Python,
- either `python3 sort.py`
- OR `python sort.py`

# How to Use
Select the sort you would like. Speed adjustments on the bottom as well as array size scaling.
